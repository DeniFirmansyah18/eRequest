# eRequest
    Cara menjalankan:
    1. git clone https://github.com/DeniFirmansyah18/eRequest.git
    2. cd eRequest
    3. npm install
    4. npm run dev
    5. buka xampp
    6. php artisan migrate
    7. php artisan db:seed
    8. buka terminal baru didalam folder eRequest
    9. php artisan serve
